Projects

Hospital Management System

Hospital Management System Overview: This project was designed to simplify and automate the management of a hospital's daily operations.
Technologies Used: Java, JSP, Servlet, SQL Database.
Key Features: Admin dashboard to manage doctors, patients, and appointments. Secure login system for administrators (with credentials stored in the database). Functionalities to add, update, and delete patient and doctor records. Organized data for easy retrieval and updates.
Learning Outcome: This project enhanced my understanding of building database-driven web applications and implementing secure admin functionalities.

Music Streaming Website

Music Streaming Website Overview: This project is a feature-rich music streaming platform designed to provide users with a seamless and engaging music experience. It allows users to explore, play, and manage their favorite tracks. Technologies Used: Frontend: React.js for building a dynamic and interactive user interface. Backend: Node.js and Express.js for handling server-side operations. Database: MongoDB for efficient data storage and retrieval. Authentication: JSON Web Tokens (JWT) for secure user authentication. Key Features: Browse and search for songs and playlists. User registration and login system for personalized experiences. Create, edit, and share custom playlists. Real-time music playback with controls (play, pause, next, previous). Responsive design for both desktop and mobile users. Learning Outcome: This project deepened my understanding of full-stack development, state management in React, and backend API integration. It also enhanced my knowledge of secure authentication and database design.
